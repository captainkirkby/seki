Seki Documentation

Configuration

Out of the box, DefaultConfig.js contains a usable set of properties for testing and experimentation.

For deployment the file ConfigCustom.js should be copied into a new file called Custom.js, and the properties adjusted to requirements. Config.js should not be publicly visible.

(DefaultConfig.js and Config.js define the same set of properties, if Config.js doesn't exist then DefaultConfig.js is used).

Admin Console

A POST to URI pattern /admin/{command} will call an admin function. If {command} is unknown, a 404 will be returned.

bake
persist graphs to filesystem as Turtle files following the path structure of the graph's URI
unbake
read graphs from filesystem

handled by functions in /admin/Admin.js
