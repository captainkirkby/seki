asynchronous tree walk root - root path fileCb - callback function (file, next) called for each file -- the callback must call next(falsey) to continue the iteration, or next(truthy) to abort the iteration. doneCb - callback function (err) called when iteration is finished or an error occurs.

example:

forAllFiles('~/', function (file, next) { sys.log(file); next(); }, function (err) { sys.log("done: " + err); });
